# Tutorial_Python-Basics

Basics of Python3 - Syntax introduction.

To run script in terminal: python3 + filename.py

Fundamental building blocks of Python:

Numeric Types: <br>
integers (int) for whole numbers<br>
floats (float) for decimal numbers<br>
Strings (str): <br>
to create and manipulate text <br>
Booleans (bool): <br>
True and False values and use them for comparisons.<br>
Operators: <br>
arithmetic operators (\*) <br>
comparison (==, <), and boolean (and, or) operators.<br>
Type Conversion (Casting): <br> convert between different data types, like turning a string into a number.
